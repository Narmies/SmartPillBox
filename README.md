💊 Smart Pill Box — ჭკვიანი წამლის ყუთი
**ილიას სახელმწიფო უნივერსიტეტი — კომპიუტერული ინჟინერიის სადამამთავრებლო (Capstone) პროექტი, 2026**

## 📌 პრობლემის აღწერა

მედიკამენტების დანიშნულებისამებრ მიღების დარღვევა მნიშვნელოვანი გამოწვევაა ჯანდაცვაში — განსაკუთრებით ხანდაზმული პაციენტებისა და ქრონიკული დაავადებების მქონე პირებისთვის. დოზის გამოტოვება ან არასწორი დროულობა ამცირებს მკურნალობის ეფექტურობას და შეიძლება გამოიწვიოს სერიოზული გართულებები. ბაზარზე არსებული სმარტ წამლის დისპენსერები ან ძვირია, ან ტექნიკურად რთული, ან საჭიროებს სპეციალურ მობილურ აპლიკაციას.

### 📱 ვებ კონფიგურაციის ინტერფეისი

მოწყობილობა მუშაობს captive-პორტალის რეჟიმში — აპლიკაცია საჭირო არ არის:

1. მოწყობილობა ირთვება და ავრცელებს "SmartPillBox-Ayarlar" Wi-Fi ჰოტსპოტს
2. მომხმარებელი უერთდება ტელეფონით → ბრაუზერი ავტომატურად იხსნება
3. მომხმარებელი ადგენს:
   - განყოფილება 1-ის დროს (სავალდებულო)
   - განყოფილება 2-ის დროს (სავალდებულო)
   - სახლის Wi-Fi მონაცემებს (სურვილისამებრ — Telegram-ისთვის)
   - Telegram Chat ID-ს (სურვილისამებრ — დისტანციური შეტყობინებებისთვის)
4. ფორმის გაგზავნისას:
   - სახურავები იხურება და იბლოკება
   - RTC სინქრონიზდება ტელეფონის მიმდინარე დროსთან
   - სისტემა გადადის აქტიური მონიტორინგის რეჟიმში

თუ Wi-Fi მონაცემები ცარიელია, მოწყობილობა მუშაობს ლოკალურ რეჟიმში — მხოლოდ ხმოვანი/ვიზუალური შეხსენებებით, Telegram-ის გარეშე.

## 🚀 გამართვის ინსტრუქციები

### 1. ფირმვეარის ჩატვირთვა

1. დააინსტალირეთ Arduino IDE 2.x
2. დაამატეთ ESP32 დაფის მხარდაჭერა:
   `https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json`
3. დააინსტალირეთ ბიბლიოთეკები Tools → Manage Libraries-ის მეშვეობით:
   - RTClib by Adafruit
   - ESP32Servo by Kevin Harrington
   - UniversalTelegramBot by Brian Lough
   - ArduinoJson by Benoit Blanchon
4. აირჩიეთ დაფა: Tools → Board → ESP32 Dev Module
5. ატვირთეთ firmware/smart_pillbox.ino

### 2. პირველი გამოყენება

1. ჩართეთ USB-ის მეშვეობით (5V)
2. დააკავშირეთ ტელეფონი "SmartPillBox-Ayarlar" Wi-Fi-ს
3. ბრაუზერი ავტომატურად გაიხსნება → შეავსეთ განრიგის ფორმა
4. გააგზავნეთ → სისტემა მზადაა

### 3. Telegram Chat ID-ის მიღება

1. მოძებნეთ @userinfobot Telegram-ში → გაუგზავნეთ /start
2. ბოტი გიპასუხებთ თქვენი Chat ID-ით
3. შეიყვანეთ ის ვებ ფორმაში (სურვილისამებრ)
