# 💊 SimpleMed - Smart Medicine Box (ჭკვიანი წამლების ყუთი)

ეს პროექტი არის ESP32-სა და RTC DS3231 მოდულზე ბაზირებული, ორ სლოტიანი **ჭკვიანი წამლების ყუთი**. მოწყობილობის კონფიგურაცია ხდება მარტივი ვებ-ინტერფეისის საშუალებით და ინტეგრირებული აქვს Telegram შეტყობინებების სისტემა.

## 🚀 ფუნქციები
- **ორენოვანი მხარდაჭერა:** ვებ-ინტერფეისი და Telegram შეტყობინებები მუშაობს როგორც ინგლისურ, ასევე ქართულ ენაზე.
- **სერვო ძრავების უსაფრთხო კონტროლი:** გაჭედვის საწინააღმდეგო სპეციალური კუთხეები ($50^\circ$ და $55^\circ$) და თავსახურის ავტომატური დაკეტვა 120 წამის შემდეგ.
- **RTC DS3231 დროის კონტროლი:** რეალური დროის მოდული, რომელიც ინარჩუნებს დროს დენის გათიშვის შემთხვევაშიც კი.
- **Telegram შეტყობინებები:** შეხსენება წამლის დროის მოსვლისას, დასტური მიღებისას და გაფრთხილება მოულოდნელი გათიშვის (Power Loss) შესახებ.
- **აქტიური განგაში:** წყვეტილი ხმოვანი სიგნალი (Buzzer) და LED ინდიკატორი, რომელიც ითიშება ღილაკზე ხელის დაჭერით.
- **მკურნალობის კურსის ხანგრძლივობა:** დღეების რაოდენობის განსაზღვრა (მაგ. 7 დღის შემდეგ შეხსენებები ავტომატურად ჩერდება).

## 🛠️ საჭირო კომპონენტები
- ESP32 Development Board
- Real Time Clock RTC DS3231 მოდული
- 2x SG90 სერვო ძრავი (თავსახურებისთვის)
- Buzzer (აქტიური ხმოვანი სიგნალი) და LED ნათურა
- Push Button (ღილაკი - Interrupt მხარდაჭერით)

## 📌 პინების განაწილება (Pinout)
- **Buzzer:** Pin 5
- **LED:** Pin 2
- **Button:** Pin 15 (Internal Pull-Up)
- **Servo 1 (სლოტი 1):** Pin 18
- **Servo 2 (სლოტი 2):** Pin 19
- **I2C (RTC მოდულისთვის):** ESP32-ის სტანდარტული SDA/SCL პინები

## 💻 დაყენების ინსტრუქცია
1. გახსენით `firmware/SimpleMed.ino` ფაილი Arduino IDE-ში.
2. დარწმუნდით, რომ დაინსტალირებული გაქვთ შემდეგი ბიბლიოთეკები: `UniversalTelegramBot`, `RTClib`, `ESP32Servo`.
3. კოდში `YOUR_TELEGRAM_BOT_TOKEN_HERE`-ს ნაცვლად ჩაწერეთ თქვენი ბოტის რეალური ტოკენი.
4. ჩატვირთეთ კოდი ESP32-ზე.
5. ტელეფონით ან კომპიუტერით დაუკავშირდით Wi-Fi ქსელს სახელით: **SmartPillBox-Setup**.
6. ბრაუზერში გახსენით მისამართი `192.168.4.1`, აირჩიეთ ენა, მიუთითეთ საათები, თქვენი სახლის Wi-Fi-ს მონაცემები და Telegram Chat ID.
